# 🚀 AI Resume Analyzer

An AI-powered Resume Analyzer that evaluates resumes against job descriptions using NLP and provides ATS score, skill gap analysis, and suggestions.

## 🔥 Features

* Resume parsing (PDF/DOCX)
* Job Description matching
* ATS Score calculation
* Skill extraction
* Skill gap visualization
* Suggestions for improvement
* Downloadable PDF report

## 🛠 Tech Stack

* Python
* Streamlit
* Scikit-learn (TF-IDF, Cosine Similarity)
* Matplotlib
* ReportLab

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📸 Output

* ATS Score
* JD Match %
* Skills Found
* Missing Skills
* Suggestions
